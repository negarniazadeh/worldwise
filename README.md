# WorldWise

A travel tracking app built with React — currently in progress.

## Features (Completed)

- Multi-page navigation with React Router
- Home, Pricing, Product, and Login pages
- Cities list fetched from JSON Server (fake API)
- Countries section showing visited countries
- App layout with sidebar and map view

## Tech Stack

- React.js
- React Router
- useState / useEffect
- JSON Server (fake API)
- Vite
- CSS3

## Getting Started

```bash
npm install
npm run server
npm run dev
```

> 🚧 This project is currently under active development.
